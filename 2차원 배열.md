- 2차원 배열

  2차원 배열이란 배열의 배열임.

  - 2차원 배열 생성 방법

    - 정수를 4개씩 담을 수 있는 배열이 3개 생성됨.

    ```java
    int[][] array4 = new int[3][4];
    ```

  - 2차원 배열에 값을 저장하는 방법

    - 만약 array4[1] = 10 ; 이렇게 사용하면 오류 발생.
    - array4[1]은 또 다른 1차원 배열을 가리킬 수 있는 참조형 변수이기 때문에 값을 담을 수는 없음.

    ```java
    array4[0][0] = 10;
    ```

  - 가변 크기의 2차원 배열을 생성하는 방법

  ```java
  int[][] array5 = new int[3][];
  
  array5[0] = new int[1];
  array5[1] = new int[2];
  array5[2] = new int[3];
  ```

  - 선언과 동시에 초기화하는 방법

  ```java
  int[][] array6 = {{1}, {2, 3}, {4, 5, 6}};
  ```
