- 삼항 연산자

  - 조건식 ? 피연산자1 : 피연산자2

    - 조건식의 연산결과가 true이면, 결과는 피연산자 1이고, 조건식의 연산결과가 false이면 결과는 피연산자2

    ```java
    int b1 = ( 5 > 4 ) ? 50 : 40;
    ```

    - 피연산자1, 피연산자2 에는 주로 값이 오지만, 경우에 따라 연산식이 올 수 있음.
    - 삼항 연산자가 익숙하지 않다면 if문으로 바꾸어 이용해도 됨.

    ```java
    int b2 = 0;
    if(5 > 4) {
    		b2 = 50;
    }else{
    		b2 = 40;
    }
    ```
